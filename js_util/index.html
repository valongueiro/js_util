Example #2 | Serializing and array that contains several objects

/*
 *  An array of objects with multiple values
 *  
 * 	In this case serialize(); assumes that you pass as a paramenter an array containing objects with two default values:
 *  'name' and 'value'   
 */
var arrayPeopleNames = [
	{name: "name", value: "Andre"},
	{name: "last", value: "Valongueiro"},
	{name: "programming", value: "JavaScript"},
	{name: "country", value: "Brazil"},
	{name: "blog", value: "www.valongueiro.blog.br"}	
]
Serialize arrayPeopleNames object
